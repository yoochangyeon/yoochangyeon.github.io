# Blog Customization Requirements

## User Requirements
1. **Blog Name**: "개발하는 유에녹" (Developing Enoch)
2. **Author**: 유창연 (Enoch) - "맨땅에 헤딩 전문가"
3. **Homepage**: Blog posts + author profile (no marketing content)
4. **Navigation**: Only 2 items - "글" (Posts) and "카테고리" (Categories)
5. **Focus**: Reading-first blog design, remove all unnecessary pages

## Pages to Remove
- All `/homes/*` variants (SaaS, Startup, Mobile App, Personal)
- All `/landing/*` pages
- `/services`, `/pricing`, `/about`, `/contact`
- `/terms`, `/privacy`

## Configuration Changes
- Blog name: "개발하는 유에녹"
- Language: Korean (ko)
- Description: "기술과 개발에 대한 생각을 나누는 공간"
- Author: 유창연 (Enoch)
- Tagline: "맨땅에 헤딩 전문가"

## Design Principles
- Content-first approach
- Minimal navigation (2 items only)
- Profile integrated into homepage
- Category-based organization
- High readability optimization
