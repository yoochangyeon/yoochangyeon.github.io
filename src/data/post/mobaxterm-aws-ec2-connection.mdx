윈도우에서 AWS EC2 서버에 접속하기 위해서 MobaXterm을 사용하였다.

다운로드는 공식 홈페이지([https://mobaxterm.mobatek.net/](https://mobaxterm.mobatek.net/))에서 받을 수 있다.

먼저 AWS EC2에서 포트를 바꿔주어야 한다.

![](~/assets/images/posts/mobaxterm-aws-ec2-connection/image-0.png)

아래 바를 오른쪽으로 옮기면 보안 그룹이 나오는데, 보안그룹을 클릭해준다.

![](~/assets/images/posts/mobaxterm-aws-ec2-connection/image-1.png)

이후 인바운드 규칙 - 인바운드 규칙 편집 클릭

![](~/assets/images/posts/mobaxterm-aws-ec2-connection/image-2.png)

규칙 추가를 눌러주고 포트는 8080, 소스는 0.0.0.0/0 으로 선택해준다. 이후 규칙 저장.

MobaXterm을 실행시켜준다.

![](~/assets/images/posts/mobaxterm-aws-ec2-connection/image-3.png)

MobaXterm 실행 후 Session 버튼 클릭

![](~/assets/images/posts/mobaxterm-aws-ec2-connection/image-4.png)

이후 SSH 클릭

![](~/assets/images/posts/mobaxterm-aws-ec2-connection/image-5.png)

Remote host에는 AWS EC2 인스턴스에 있는 퍼블릭 DNS(IPv4) 주소를 입력

![](~/assets/images/posts/mobaxterm-aws-ec2-connection/image-6.png)

이후 아래에 Advanced SSH settings 탭에 User private key를 체크해주고, 이전에 다운받았던 키 페어를 찾아 클릭해준다.

![](~/assets/images/posts/mobaxterm-aws-ec2-connection/image-7.png)

login as에 ubuntu를 입력해주었다.

![](~/assets/images/posts/mobaxterm-aws-ec2-connection/image-8.png)

위에 화면처럼 mobaXterm을 이용하여 AWS EC2 서버에 접속한것을 확인할 수 있다,
