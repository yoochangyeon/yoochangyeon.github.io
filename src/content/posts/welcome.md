## 블로그를 시작하며

안녕하세요! Astro 프레임워크로 구축한 기술 블로그에 오신 것을 환영합니다.

이 블로그는 **가독성**을 최우선으로 설계되었습니다. 깔끔한 디자인과 함께 기술 문서를 읽기 편하게 만들었습니다.

## 주요 특징

이 블로그는 다음과 같은 특징을 가지고 있습니다:

1. **뛰어난 가독성**: 70자 너비의 최적화된 텍스트 레이아웃
2. **반응형 디자인**: 모바일과 데스크탑 모두에서 완벽한 가독성
3. **코드 하이라이팅**: Shiki를 활용한 아름다운 코드 블럭
4. **빠른 로딩**: Astro의 정적 사이트 생성으로 초고속 페이지 로드

## 코드 블럭 예시

아래는 JavaScript 코드 예시입니다:

\`\`\`javascript
function greet(name) {
  return \`Hello, \${name}!\`;
}

const message = greet("World");
console.log(message);
\`\`\`

Python 코드도 깔끔하게 표시됩니다:

\`\`\`python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# 처음 10개의 피보나치 수
for i in range(10):
    print(fibonacci(i))
\`\`\`
